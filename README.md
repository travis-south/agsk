# agsk

Personal agent skills, installable with [skills.sh](https://skills.sh).

## Skills

- `commit-and-push` — commit intended changes with a branch-derived Conventional Commit scope, then push and verify the current branch.
- `ship-spec` — implement an approved spec on its own branch, update tracker state, independently review each work item, and open a linked draft pull request.

## Install

```bash
npx skills@latest add travis-south/agsk
```
